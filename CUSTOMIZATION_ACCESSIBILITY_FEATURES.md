# Customization and Accessibility Features

## Overview

This application includes customization and accessibility features to improve user experience.

## Customization

### Theme
- Light mode (default)
- Dark mode
- Toggle icon in navigation

### Language Support
- English
- French
- Punjabi
- Hindi

### Font Sizes
- Small
- Medium (default)
- Large

## Accessibility

### WCAG 2.1 AA Compliance
- Proper color contrast
- Semantic HTML
- ARIA labels
- Keyboard navigation

### Features
- Screen reader support
- Keyboard navigation
- Reduced motion option
- High contrast mode

## Settings

Access settings through the navigation menu to customize theme, language, and accessibility preferences.

All settings are saved to browser storage and persist across sessions.
